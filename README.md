# LeanFlow 🚀

**LeanFlow** is a minimalist tool designed for Linux users with low-end hardware. 
It helps you monitor and optimize system resources with zero overhead.

## 🛠 Features
- **Smart Kill:** Automatically stop heavy background tasks.
- **Resource Monitor:** See what eats your RAM in real-time.
- **Turbo Mode:** One-click optimization for coding or gaming.
